## Описание

Небольшое приложение, в котором можно просматривать, добавлять и редактировать заявки. Все взаимодействия происходят через API бэкенда.

Я использовал TypeScript, Next.js, Redux Toolkit и SCSS модули. Все задачи, поставленные в ТЗ, выполнил.

- Подгрузка необходимых данных с бэка. Инициализация происходит после перехода по роуту /requests.
- Добавление комментариев.
- Изменение статуса и исполнителя реализовано с помощью кастомного селекта. Он открывается при нажатии на сам статус/исполнителя внутри заявки.
- Все изменения, совершенные в заявке, сохраняются по кнопке "Сохранить".
- Дополнительно добавлены отображения ошибок в интерфейсе.

## Запуск

Команды для того, чтобы развернуть проект:
1. npm install — установка зависимостей,
2. npm run dev — запуск в режиме разработки,
3. npm run build — сборка продовой версии,
4. npm run start — запуск продовой версии.
